# Changelog

## 0.2.0 (2026-04-09)

### Added
- Unified gateway to all AnyAI sub-packages (anycv, anyocr, anyllm, anyml, anynlp, anydeploy, tableai, traincv, anyrobo)
- Local-first / edge AI support
- Async API support
- CLI interface via `anyai` command
- Jupyter notebook integration with rich HTML output
- Rich TUI output for terminal usage
- Proxy functions for image, text, and pipeline operations
- Comprehensive configuration management
- Version info utility across all sub-packages

### Fixed
- Improved error messages when optional sub-packages are not installed
- Graceful handling of missing dependencies
